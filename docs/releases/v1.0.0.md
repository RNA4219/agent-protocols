## ハイライト

- low / medium は自動承認、high / critical は人間承認
- `approvalDeadline` と `pending` の境界を明文化
- `Evidence` を `Published` 状態の不変記録として扱う
- ID prefix を `IC` `TS` `AC` `PG` `EV` に固定
- JSON Schema と semantic validation を分離
